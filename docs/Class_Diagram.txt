MVVM Architecture Class Structure
------------------------------------------------------------------------

1. MODELS (Data Layer)
========================================================================

  ┌─────────────────────────────────┐
  │ «interface» User                │
  ├─────────────────────────────────┤
  │ + uid: string                   │
  │ + email: string                 │
  │ + displayName: string           │
  │ + role: "admin"|"manager"       │
  │ + createdAt: Date               │
  │ + lastLoginAt: Date             │
  └─────────────────────────────────┘

  ┌─────────────────────────────────┐
  │ «interface» Employee            │
  ├─────────────────────────────────┤
  │ + id: string                    │
  │ + employeeId: string            │
  │ + firstName: string             │
  │ + lastName: string              │
  │ + email: string                 │
  │ + phone: string                 │
  │ + department: Department        │
  │ + designation: string           │
  │ + dateOfJoining: Date           │
  │ + dateOfBirth: Date             │
  │ + gender: string                │
  │ + address: Address              │
  │ + bankDetails: BankDetails      │
  │ + basicSalary: number           │
  │ + status: string                │
  │ + createdAt: Date               │
  │ + updatedAt: Date               │
  └─────────────────────────────────┘
           │ contains
           ▼
  ┌──────────────────┐  ┌──────────────────┐
  │ «interface»      │  │ «interface»      │
  │ Address          │  │ BankDetails      │
  ├──────────────────┤  ├──────────────────┤
  │ + street: string │  │ + bankName: str  │
  │ + city: string   │  │ + accountNumber  │
  │ + state: string  │  │ + ifscCode: str  │
  │ + zipCode: string│  │ + accountHolder  │
  │ + country: string│  └──────────────────┘
  └──────────────────┘

  ┌─────────────────────────────────┐
  │ «interface» Attendance          │
  ├─────────────────────────────────┤
  │ + id: string                    │
  │ + employeeId: string            │
  │ + date: Date                    │
  │ + checkIn: string               │
  │ + checkOut: string              │
  │ + status: AttendanceStatus      │
  │ + hoursWorked: number           │
  │ + overtime: number              │
  └─────────────────────────────────┘

  ┌─────────────────────────────────┐
  │ «interface» Payroll             │
  ├─────────────────────────────────┤
  │ + id: string                    │
  │ + employeeId: string            │
  │ + month: number                 │
  │ + year: number                  │
  │ + basicSalary: number           │
  │ + allowances: Allowances        │
  │ + deductions: Deductions        │
  │ + grossSalary: number           │
  │ + totalDeductions: number       │
  │ + netSalary: number             │
  │ + status: PayrollStatus         │
  └─────────────────────────────────┘
           │ contains
           ▼
  ┌──────────────────┐  ┌──────────────────┐
  │ «interface»      │  │ «interface»      │
  │ Allowances       │  │ Deductions       │
  ├──────────────────┤  ├──────────────────┤
  │ + hra: number    │  │ + pf: number     │
  │ + da: number     │  │ + esi: number    │
  │ + ta: number     │  │ + tax: number    │
  │ + medical: number│  │ + profTax: number│
  │ + special: number│  │ + loan: number   │
  │ + overtime: num  │  │ + other: number  │
  └──────────────────┘  └──────────────────┘

  ┌─────────────────────────────────┐
  │ «interface» Payslip             │
  ├─────────────────────────────────┤
  │ + id: string                    │
  │ + payrollId: string             │
  │ + employeeId: string            │
  │ + earnings: PayslipEarning[]    │
  │ + deductions: PayslipDeduction[]│
  │ + grossEarnings: number         │
  │ + totalDeductions: number       │
  │ + netPay: number                │
  │ + generatedAt: Date             │
  └─────────────────────────────────┘


2. SERVICES (Data Access Layer)
========================================================================

  ┌────────────────────────────────┐
  │ authService                     │
  ├────────────────────────────────┤
  │ + signIn(email, pass): User    │
  │ + logOut(): void               │
  │ + onAuthChange(cb): Unsub      │
  │ + getUserProfile(uid): User    │
  └────────────────────────────────┘

  ┌────────────────────────────────────┐
  │ employeeService                     │
  ├────────────────────────────────────┤
  │ + getAllEmployees(): Employee[]     │
  │ + getEmployeeById(id): Employee    │
  │ + createEmployee(data): string     │
  │ + updateEmployee(id, data): void   │
  │ + deleteEmployee(id): void         │
  │ + getActiveEmployeeCount(): number │
  └────────────────────────────────────┘

  ┌────────────────────────────────────────┐
  │ attendanceService                       │
  ├────────────────────────────────────────┤
  │ + getAllAttendance(): Attendance[]      │
  │ + getAttendanceByEmployee(id): Att[]   │
  │ + getAttendanceByMonth(m, y): Att[]    │
  │ + createAttendance(data): string       │
  │ + getAttendanceSummary(...): Summary   │
  └────────────────────────────────────────┘

  ┌────────────────────────────────────────┐
  │ payrollService                          │
  ├────────────────────────────────────────┤
  │ + getAllPayrolls(): Payroll[]           │
  │ + createPayroll(data): string          │
  │ + updatePayrollStatus(id, st): void    │
  │ + calculateSalaryLocal(req): Result    │
  │ + calculateSalaryAPI(req): Result      │
  │ + calculateTaxAPI(req): TaxResult      │
  └────────────────────────────────────────┘


3. VIEWMODELS (Logic Layer)
========================================================================

  ┌─────────────────────────────────────┐
  │ useAuthViewModel (Zustand Store)     │
  ├─────────────────────────────────────┤
  │ - user: User | null                 │
  │ - isAuthenticated: boolean          │
  │ - isLoading: boolean                │
  ├─────────────────────────────────────┤
  │ + login(email, pass): Promise       │
  │ + logout(): Promise                 │
  │ + initAuthListener(): Unsub         │
  └─────────────────────────────────────┘
           │ uses
           ▼
      authService

  ┌─────────────────────────────────────┐
  │ useEmployeeViewModel (Zustand)       │
  ├─────────────────────────────────────┤
  │ - employees: Employee[]             │
  │ - selectedEmployee: Employee | null │
  │ - searchQuery: string               │
  │ - filterDepartment: string          │
  ├─────────────────────────────────────┤
  │ + fetchEmployees(): Promise         │
  │ + addEmployee(data): Promise        │
  │ + editEmployee(id, data): Promise   │
  │ + removeEmployee(id): Promise       │
  │ + getFilteredEmployees(): Employee[]│
  └─────────────────────────────────────┘
           │ uses
           ▼
      employeeService

  (Similar patterns for Attendance, Payroll, Dashboard ViewModels)

  │ Sidebar, Header   │
  └───────────────────┘
